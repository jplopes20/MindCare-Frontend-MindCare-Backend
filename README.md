# MindCare — Frontend Demo

Protótipo frontend interativo para apresentação acadêmica: UI de plataforma de saúde mental com Login, Dashboards (Paciente e Profissional), Teleconsulta visual e Prontuário Eletrônico.

Run

```bash
cd Projeto_4
npm install
npm run dev
```

O projeto é um protótipo visual — funcionalidades clínicas reais (armazenamento seguro, backend, sinalização WebRTC, autenticação OAuth) não estão implementadas aqui. Use como demo para apresentação.

Publicar no GitHub

1. Crie um repo no GitHub (por ex. `mindcare-frontend`).
2. No terminal do projeto:

```bash
git init
git add .
git commit -m "Initial MindCare frontend demo"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/mindcare-frontend.git
git push -u origin main
```

Se quiser, posso ajudar a formatar o README ou preparar a versão para deploy (Netlify / Vercel).
